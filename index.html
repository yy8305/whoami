맛집 리뷰 어플

주요기능

- 회원가입 : 회원가입한 사람들만 어플 이용가능
- 맛집리스트 : 맛집들을 스크롤을 통해 확인 할 수 있으며 카테고리, 가게명 으로 검색하여 확인 가능
- 맛지도 : 맛집들의 위치를 지도를 통해 확인 할 수 있으며 현재 위치도 확인 가능
- 예약하기 : 날짜, 시간, 인원을 입력하여 예약 할 수 있으며 예약한 내용은 예약 현황 메뉴에서 확인 가능
- 후기작성 : 별점을 통하여 맛집들을 평가 할 수 있으며 작성된 후기들은 맛집 상세정보에서 확인 가능
- 즐겨찾기 : 맛집들을 즐겨찾기하여 확인 할 수 있으며 즐겨찾기 메뉴에서 한번에 확인 가능

어플 구조

- DB는 MYSQL을 이용
- API서버는 PYTHON FLASK로 만들어졌으며 사용자 요청에 대한 명령 처리후 응답
- JAVA단에서 API서버와 통신하기 위해 Retrofit, Picasso 라이브러리 사용

Using stack:

  • ANDROID JAVA
  • PYTHON FLASK
  • MYSQL
  • Retrofit
  • Picasso
  • Google Map API
